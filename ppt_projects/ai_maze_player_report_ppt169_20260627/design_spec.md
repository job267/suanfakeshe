## XI. Technical Constraints Reminder

### SVG Generation Must Follow:

1. viewBox: `0 0 1280 720`
2. Background uses `<rect>` elements
3. Text wrapping uses `<tspan>` (`<foreignObject>` FORBIDDEN)
4. Transparency uses `fill-opacity` / `stroke-opacity`; `rgba()` FORBIDDEN
5. FORBIDDEN: `mask`, `<style>`, `class`, `foreignObject`
6. FORBIDDEN: `textPath`, `animate*`, `script`
7. XML reserved chars in text MUST be escaped as `&amp;` `&lt;` `&gt;` `&quot;` `&apos;`.
8. Use only icons listed in Section VI and `spec_lock.md`.
9. Use only images listed in Section VIII and `spec_lock.md`.

### PPT Compatibility Rules:

- `<g opacity="...">` FORBIDDEN; set opacity on each child element individually.
- Inline attributes only; external CSS and `@font-face` FORBIDDEN.
- Avoid excessive text per card; split dense code details into tables and annotations.
